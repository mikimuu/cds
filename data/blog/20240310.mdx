# 好きだったポッドキャストが終わってしまった
起きてベッドでTwitter見てたら「今後の更新は未定です」ツイートが流れてきた。
<blockquote class="twitter-tweet"><p lang="ja" dir="ltr">【底辺文化系トークラジオ「二九歳までの地図」の更新につきまして】この度は正式なアナウンスをせずにいた事お詫び申し上げます。取り急ぎ今後の更新は未定となります。（1/7）</p>&mdash; 底辺文化系トークラジオ「二九歳までの地図」 (@29tizu) <a href="https://twitter.com/29tizu/status/1766447998484316235?ref_src=twsrc%5Etfw">March 9, 2024</a></blockquote> <script async src="https://platform.twitter.com/widgets.js" charset="utf-8"></script>

大学1年生の時にききだして、4年生くらいまで真面目に聞いていた。
「底辺文化系」のお兄さんの会話をニヤニヤしながら聞いていた。
メンバーが4人いて、それぞれの映画に対する評価が全然違うのも面白かったしたまに喧嘩みたいになる緊張感も好きだったのだけれど。

とはいえここ一年くらい更新がなかったし、メンバーのTwitterもそれぞれぼんやりなってきたところで「もう終わるんだろうな」とは感じていた。

で、ここからはぼくのはなしになるんだけど、、じゃあぼくは「二十九歳」まで底辺文化系をやれるだろうか。底辺じゃないにしても文化系でもいいし、サブカルでもいい。<br/>
なんとなく、やれない気がしている。もう「29地図」を聞いていた時みたいに、常に新しい音楽を聞いて、毎週映画館に行って、ぼんやり本読んだり漫画読んだりして、マーベルについて熱く考えたりするのはもう無理な気がする。
<br/>
それはとても怖いし悲しいことな気もするけど、別に一生「文化系」にしがみつくのもなぁ、とも思う。

## ここで"New Jeans おじさん"へ
そんなことを考えていたら、New Jeansおじさんの代表こと宇野コレマサが思い浮かんだ。

そしてその時、端的にいうと、めっちゃ尊敬した。

おじさんになっても新しい映画と音楽を追いかけ続けて、それについて発信しているなんて、、、！！！

まぁもちろん仕事なんだろうけど、すごいえらいと思った。つまり、あらゆる文化系おじさんをえらいと思ったのだ。

### New Jeansおじさんの別の可能性
じゃあ宇野コレマサみたいな人が社会人になっても、29歳を超えても、サブカルをやっている稀有なおじさんだとして、「元文化系おじさん」はどこに行ってしまったのか。

そしてこのツイートを思い出す。
<blockquote class="twitter-tweet"><p lang="ja" dir="ltr">クリエイティブ職に就かなかった（あるいは就きたかったのに就けなかった）サブカル系の人々のその後の人生について興味がある</p>&mdash; 鈴木眼鏡 (@qcuekew) <a href="https://twitter.com/qcuekew/status/1734829352330883434?ref_src=twsrc%5Etfw">December 13, 2023</a></blockquote> <script async src="https://platform.twitter.com/widgets.js" charset="utf-8"></script>

これ見た時「キッショいツイーとだな〜〜〜」と思った。でも今はぼくの前に割と切実な問題として現れている。そして宇野コレマサとは別のNew Jeansおじさんがいる気がしてきた。



実際のところNew Jeansおじさんがどんな人たちなのか知らないけど、「クリエイティブ職に就かなかったサブカル系」が、サブカルでもなんでもない普通のおじさんになってしまったあと、「サブカル系」だった時の感情とか記憶を投影できる現行の作品を見つけた時ってそりゃ興奮するだろ！！！

と、思いました。

「ぼくもそんなおじさんになりそうな気がする。だが、まだ俺に残されている。痛いサブカルおじさんへの道が。。。」

とも、思いました。

なのでまえやってたプレイリスト作る活動を今日はやってみました。粛々と、痛いサブカルおじさんへの道の第一歩ささやかに踏み出すのだ。！
