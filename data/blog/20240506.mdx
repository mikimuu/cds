# 4. センスの哲学 / 千葉雅也
会社で「読めや！」と言われたビジネス書の類の本は朝とか夜とか読むことにしている。
そして、音楽と同じように、通勤中くらいは好きな本を読むことにしている。

>餃子を食べるときに口の中でいろんな刺激が展開して、「おいしい！」と言うときには、それにまつわる思い出とかも言えるわけだけれども、その味わいの運動自体はとくに意味がない。強度的、リズム的体験なわけです。ラウシェンバーグのこの作品を見ることは、餃子を食べておいしい、というのとほとんど同じ体験なのです。
>千葉 雅也. センスの哲学 (文春e-book) (p.57). 文藝春秋. Kindle 版. 

らしい。千葉雅也の文章ではこうやって意味のわからないものが並列に置かれたりする。
それがなぜできるのか、どうやってやるのか、みたいなことを書いている。つまりセンスの哲学は制作論として読める。
「もう何も生み出せないよ、、、」と思っていた社会人１ヶ月目の僕が読み、それなりに良かった。

ネタ切れ。

さて、GWもあと1日で終わるらしい。
彼女が福岡から来てくれたので、ほとんどずっと過ごしていた。
久しぶりに実家に帰ったりもした。かなり楽しく過ごした。

GWなんてガキの頃からずっとあるはずだが、果たしてGWをちゃんと楽しめていたことはこれまでなかったんじゃないかというくらい楽しかった、充実していた、気がする。

第一には、GWは部活している中学生、バイトしている高校生、サークルでイベントをしている大学生のものではなく、社会人のためのもの。そして僕はやっとGWを享受できるようになった。労働者万歳、である。

第二に今年のGWは家族と恋人と過ごした。一人で過ごすことがほとんどなかった。人と過ごし、移動すれば大抵の休日は充実するものである。

倫理って「〇〇してはいけない」っていう形で表現される（殺人はいけません、人が嫌がることをしてはいけません）けど、それとは別に「〇〇しないことをする」って形もあり得るはず。

「浮気しないでね」って言われて、「そもそもしたことない、したいと思ったことすらない。」と返事するのではなく、「俺は浮気しないことをやっているんだ！」と返事してみる。
